#jQuery Sly Slider

Sly is a JavaScript library for advanced one-directional scrolling with item based navigation support.

* Keyboard navigation (Left/Right)
* Swipe enabled
* Desktop mouse dragging

##Dependencies##
* jQuery 1.7+
* jQuery Easing

##Dependencies##
Sly works in every desktop browser.
